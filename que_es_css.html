# CSS

¿Que es CSS y para qué sirve?

CSS, u hojas de estilo en cascada, es una tecnología utilizada en el desarrollo web. Sirve para controlar la presentación y el diseño visual de los documentos HTML.

Beneficios de CSS

CSS permite el estilo consistente de múltiples páginas dentro de un sitio web. Los cambios realizados en las reglas CSS se aplican automáticamente a todos los elementos que utilizan dichas reglas, lo que garantiza una apariencia uniforme.


Al utilizar CSS, puedes evitar la repetición. Las reglas de estilo se definen una vez y se pueden aplicar a varios elementos, lo que reduce la cantidad de código necesario y agiliza las actualizaciones.


Formas de aplicar CSS


CSS en línea (inline)

El CSS en línea se aplica directamente a un elemento HTML mediante el atributo style del elemento. Este enfoque permite aplicar estilos específicos y únicos a elementos individuales. Ventajas: Útil para aplicar estilos rápidos a un elemento específico. No se necesita un archivo externo ni una declaración de estilo general. Desventajas: Dificultad para mantener y actualizar los estilos. No es reutilizable ni eficiente para estilos repetitivos.


CSS en el encabezado (internal)

El CSS en el encabezado se coloca dentro de la etiqueta style en el head del documento HTML. Este método es útil cuando se necesita aplicar estilos personalizados a una sola página. Ventajas: Ideal para proyectos pequeños o cuando solo se necesita aplicar estilos a una página específica. Evita la necesidad de un archivo CSS externo. Desventajas: No es eficiente para grandes proyectos o para compartir estilos entre múltiples páginas. El tamaño del archivo HTML aumenta debido a la inclusión de estilos.


CSS externo

El CSS externo implica vincular un archivo CSS independiente desde la cabecera del documento HTML mediante la etiqueta . Este método es el más eficiente para proyectos grandes o cuando se desea aplicar los mismos estilos a múltiples páginas. Ventajas: Centraliza los estilos en un archivo separado, lo que facilita el mantenimiento y la reutilización. Permite que múltiples páginas compartan los mismos estilos, lo que reduce la duplicación. Mejora la eficiencia del sitio web, ya que los navegadores pueden almacenar en caché el archivo CSS. Desventajas: Si el archivo CSS no está disponible o tarda en cargarse, puede afectar la presentación de la página.
